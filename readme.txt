== Description ==
<h3>WMSG CONTACT FORM;</h3> Wordpress iletişim formu olarak kullanabileceğiniz, gelişmiş whatsapp contact form eklentisidir, kısa kod ile sayfalarınıza ve yazılarınıza çağırabilir. Tüm sayfalarda popup olarak aktif ederek kullanabilirsiniz. QR kodu okutmaya gerek kalmadan kullanıcılarınızın mesajı doğrudan tarafınıza iletilecek ve hızlı bir iletişim kurmanıza yardımcı olacaktır. 

<strong>NOT: </strong>WMSG CONTACT FORM eklentisini kullanabilmek içim API Key'inizin olması gerekmektedir.
<strong>https://wa.teknoltan.com/kayit.php</strong> adresinden API Key oluşturabilirsiniz.

== Frequently Asked Questions ==

= Eklentiyi kullanmak icin API KEY'e ihtiyaç var mi? =

Evet, eklentiyi kullanabilmek icin API KEY'iniz olmalıdır.

= Nasıl API KEY Alabilirim ? =

https://wa.teknoltan.com/kayit.php adresinden kayıt olup, API key kısmından keyinizi alabilirsiniz.

= Ekleti ayarları nasıl olmalı ? =
Wp-admin Panelde bulunan WMSG FORM ekranında Eklenti Ayarları Ve Tasarım Ayarları Bulunmaktadır.

<h3>Eklenti Ayaları</h3>
Bu kısmında bulunan Yönetici Telefon No kısmına telefon numaranızı 9053xxxxxxxxx şeklinde giriniz.
https://wa.teknoltan.com/kayit.php  adresinden almış olduğunuz API KEY'inizi giriniz.
Google reCAPTCHA v2 üzerinden oluşturduğunuz reCAPTCHA Site Key ve reCAPTCHA Secret Key bilgilerinizi giriniz. 
PopUp Formu aktif ediniz.

<h3>Tasarım Ayarları</h3>
Bu kısımda bulunan ayarlar ise form tasarım alanıdır, kendi renk ve zevkinize göre istediğiniz alanları tasarlayabilir renklendirmeler yapabilirsiniz.

== Installation ==
<ul>
<li>EKLENTİLER/YENİ EKLE kısmından wmsg yazıp aratınız ve yükleyiniz veya Ftp aracılığı ile /wp-content/plugins/ dizinine eklenti dosyalarını yükleyiniz. </li>

<li>Eklentiler kısmından ekletiyi etkinleştiriniz.</li>

<li>Yönetici Telefon No kısmına numaranızı 9053xxxxxxxxx şeklinde giriniz.</li>

<li>Kredi Ekle butonu ile kredi ekleme sayfasından kredi ekleyin ve WMSG API KEY 'inizi giriniz.</li>

<li>Google reCAPTCHA V2.0 oluşturun SITE KEY ve SECRET KEY alanlarına giriniz.</li>

<li>Kısa Kod : <strong>[WMSG]</strong> ile istediğiniz alana çağırarak kullanabilirsiniz.</li>
</ul>
